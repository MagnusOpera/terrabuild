---
title: "test"
---
Runs the `test` script (`pnpm run test`) across targeted workspaces.
```
@pnpm test {
    args = "--runInBand"
}
```
### Capabilities

| Capability | Info |
|------------|------|
| Cache      | remote
| Bach      | yes

## Argument Reference
The following arguments are supported:
* `args` - (Optional) Extra script arguments.
