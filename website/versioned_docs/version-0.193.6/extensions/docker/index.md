## Available Commands
| Command | Description |
|---------|-------------|
| [dispatch](/docs/extensions/docker/dispatch) | Runs an arbitrary Docker CLI command (action name is forwarded to `docker`). |
| [build](/docs/extensions/docker/build) | Builds an image tagged with the Terrabuild hash, and pushes it in CI. |
| [push](/docs/extensions/docker/push) | Pushes the built image to the registry with a specific tag. |
