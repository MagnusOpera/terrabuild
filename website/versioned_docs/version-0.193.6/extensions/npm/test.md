---
title: "test"
---
Runs the `test` script via `npm run test`.
```
@npm test {
    args = "--runInBand"
}
```
### Capabilities

| Capability | Info |
|------------|------|
| Cache      | remote
| Bach      | no

## Argument Reference
The following arguments are supported:
* `args` - (Optional) Extra arguments after `--`.
