## Available Commands
| Command | Description |
|---------|-------------|
| [execute](/docs/extensions/fscript/execute) | Executes a `.fss` file through the `fscript` CLI with the current project as sandbox root. |

## Project Initializer
FScript extension for executing project-scoped `.fss` scripts.
```
project {
  @fscript { }
}
```
Equivalent to:
```
project {
}
```
