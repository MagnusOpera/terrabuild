---
title: "build"
---
Runs the `build` script via `npm run build`.
```
@npm build {
    args = "--workspaces"
}
```
### Capabilities

| Capability | Info |
|------------|------|
| Cache      | remote
| Bach      | no

## Argument Reference
The following arguments are supported:
* `args` - (Optional) Extra arguments after `--`.
