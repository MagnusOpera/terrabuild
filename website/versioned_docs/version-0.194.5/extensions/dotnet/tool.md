---
title: "tool"
---
Executes `dotnet tool ...` commands.
```
@dotnet tool {
    args = "install MagnusOpera.OpenApiGen"
}
```
### Capabilities

| Capability | Info |
|------------|------|
| Cache      | local
| Bach      | no

## Argument Reference
The following arguments are supported:
* `args` - (Optional) Extra arguments for `dotnet tool`.
